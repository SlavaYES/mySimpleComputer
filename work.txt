

        Адресс это номер элемента массива.
        
        Память это массив целых чисел, которые имеют вес в 4 байта, то есть в 32 бита.
        
        => Адресс не должен превышать 99, ибо есть 0-нулевая позиция и 99-ая позиция.
        
        Функции:

            Памяти: 5
                Инициализация памяти
                Установка значения памяти
                Возврат значения памяти
                Сохранения в файл
                Загрузка из файла

            Регистров: 3
                Инициализация регистра флагов
                Установка значения регистра флагов
                Возврат значения регистра флагов

            Команд: 2
                Кодирование
                Декодирование
                
        value 1/0
        flag 0-31
        registr 00010
